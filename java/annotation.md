# 注解
|注解|属于|使用位置|作用|
|---|---|---|---|
|`@Configuration`|Spring|类上|告知Spring这是一个配置类，会为Spring应用上下文提供bean|
|@Bean|Spring|方法上|表示该方法返回的对象会以bean的形式添加到Spring应用上下文中|
|||||
|@Data|Lombok|类上|自动化方法生成：生成缺失的方法，生成所有以final修饰的参数的构造器|
